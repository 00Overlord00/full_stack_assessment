Database Name: 'zoo_data'

Table Name: 'critters'

Creation Command: CREATE TABLE critters (
	                id SERIAL PRIMARY KEY NOT NULL,
	                creature VARCHAR(20) UNIQUE,
	                population INTEGER NOT NULL
	              );

Additional Notes: To put things into layman's terms, the table 'critters' has three columns: id, creature, and population.

    -id: A number assigned to each entry into the table that increments automatically with each entry.
    -creature: The creature's name.
    -population: A number randomly generated by the client side to represent the number of creatures.
